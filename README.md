Project title: Hospital Analysis Dashboard

Tools used: Power BI(DAX, Power Query)

Description:This project provides a detailed analysis of hospital data using Power BI. 
            The purpose of the project is to gain insights into hospital performance, patient count,
            and other key metrics that support better healthcare decision-making.

Key insights:Total treatment costs across departments
             Department-wise patient flow
             Identified areas with high operational costs
             Visual dashboards for fast and interactive data exploration and reporting
